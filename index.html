When weight hits 0 (Empty), prompt user to refill.

Feature 2: Posture Guardian

Technology: Ultrasonic

HC-SR04 Sensor mounted on monitor measures distance to face.

Logic: Slouch Detect

System sets Baseline Distance. If deviation > 3 seconds ‚Üí "Sit Up" Alert.

Logic: Sedentary Break

If sitting for 45 minutes straight ‚Üí "Take a Walk" Alert.
